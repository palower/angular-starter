Angular Starter Kit

The purpose of this app is provide a starting point for any Angular project and reduce the amount of effort in start-up and base level component development.

Kit details

  • This project was generated with Angular CLI version 8.3.6
  • This project uses NG-Bootstrap, a combination of Bootstrap 4 and Angular with prebuilt components and scripting.

See README.md file for more info.

Development server

Run ng serve for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.

Common tasks

Generate a new component: ng g c "component-name" --module=app

  • Add routing to app-routing.module.ts
  • Remove component reference to local scss fro .ts file
  • Add .scss to stylesheet dir and import to styles.scss

Component build structure

Components follow the pattern described below:

  • Wrap each component in a .container or .container-fluid class <div>
  • Follow the Bootstrap pattern of .row > .col-*
  • No Bootstrap layout classes used in page level, only in components
  • Allows for full-width (.container-fluid) and set-max-width (.container) components
  • ALl content should be seperated from the view (HTML) and live in the .ts (TypeScript) files
